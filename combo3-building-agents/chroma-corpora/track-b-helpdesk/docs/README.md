# Streakly Help Centre

Fictional knowledge base for **Streakly** — a habit-tracker app. Track B of Combo 3's running artefact uses this as the corpus for a helpdesk agent that answers user questions and escalates to a human when appropriate.

Every article in this directory corresponds to a question we hear often. The agent's job is to match an incoming question to the right article(s), summarise the answer, and cite the source. When the agent can't confidently find an answer, it should escalate.

## About Streakly (for context)

Streakly is a mobile habit-tracker available on iOS and Android. Users define habits they want to build (e.g. "drink 2L of water", "meditate 10 min"), mark them done each day, and see their streak. Free tier covers 3 active habits; Streakly Plus ($4.99/month) removes the limit and adds widgets, insights, and cross-device sync.

## When the agent should escalate to a human

The agent should **always** escalate for:

- Billing disputes and refund requests over $20.
- Account recovery when the user can't access their email.
- Any report of suspicious activity on a user's account.
- Mentions of a bug that could be affecting multiple users.
- Requests to delete a child's account (legal review needed).
- Anything involving legal, press, or complaints.

The agent should **prefer to escalate** (but can try the article first) for:

- Questions about a specific user's billing history (we can't see it).
- Complaints about the product experience, if the tone is frustrated.
- Questions that combine multiple topics.
